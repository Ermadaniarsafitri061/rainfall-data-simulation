# 🌧️ Rainfall Data Simulation using Inverse Transform Method

## 📌 Overview
This project aims to generate synthetic rainfall data for South Lampung using statistical methods.

## ⚙️ Methods
- Inverse Transform Method
- Log-Normal Distribution
- Kolmogorov-Smirnov Test

## 📊 Process
- Performed statistical analysis (mean, skewness, kurtosis)
- Modeled rainfall distribution using Log-Normal
- Generated synthetic data using inverse transform
- Validated model using K-S test

## 📈 Results
- Synthetic data closely matches real rainfall patterns
- K-S test p-value ≈ 0.9891 (good fit)

## 🛠 Tools
- Python / Statistics
